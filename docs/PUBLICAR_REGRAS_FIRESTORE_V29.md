# Publicação das regras do Firestore — v29

A sincronização do inventário depende da regra da coleção `inventarioItens`.

## Pelo Console Firebase

1. Abra o projeto `app-manutencao-2169f`.
2. Acesse **Firestore Database**.
3. Abra a aba **Regras**.
4. Substitua o conteúdo pelo arquivo `firestore.rules` desta versão.
5. Clique em **Publicar**.

Sem esta publicação, a tela poderá exibir erro de permissão ao salvar ou carregar imagem e saldo.
