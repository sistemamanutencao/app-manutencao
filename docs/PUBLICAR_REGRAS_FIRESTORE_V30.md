# Publicar regras do Firestore — v30

1. Abra o projeto no Firebase Console.
2. Acesse **Firestore Database**.
3. Abra a aba **Regras**.
4. Substitua o conteúdo atual pelo arquivo `firestore.rules` incluído na v30.
5. Clique em **Publicar**.
6. Recarregue o aplicativo com `Ctrl + F5`.

A v30 adiciona permissão para o documento `inventarioEstrutura/principal`, usado para sincronizar andares, ambientes e áreas internas.
