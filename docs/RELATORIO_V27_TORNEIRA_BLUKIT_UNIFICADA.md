# Relatório v27 — Torneira Blukit unificada

## Alteração

O inventário passou a utilizar apenas o cadastro:

**Torneira temporizada (de mesa) Blukit**

O cadastro genérico de torneira temporizada foi removido e todas as 22 ocorrências por ambiente foram convertidas para o cadastro Blukit, totalizando 44 unidades instaladas.

## Preservação de dados locais

Ao abrir a nova versão, eventual imagem ou saldo anteriormente salvo no cadastro genérico é migrado para o cadastro Blukit quando o cadastro Blukit ainda não possui esse dado.

## Resultado da base

- 4 andares;
- 27 áreas finais;
- 12 modelos de item;
- 150 registros de item por ambiente;
- 265 unidades instaladas.
