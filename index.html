Ativos e QR Code

Cadastre equipamentos, ambientes e patrimônios para vincular cada OS a um histórico rastreável.

Como usar o QR Code

Depois de cadastrar um ativo ou local, o app gera o link do QR Code e permite imprimir uma etiqueta simples com código, nome, localização e endereço.

Use o leitor para consultar histórico, abrir nova OS vinculada ou conferir reincidência daquele ponto.
